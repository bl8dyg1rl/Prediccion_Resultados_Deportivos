# Prediccion_Resultados_Deportivos
Proyecto Final para la asignatura Análisis Avanzado de Datos en la que se usó una base de datos de partidos desde 2000 al 2025 de la Premier League de Inglaterra para aplicar las herramientas de Series de Tiempo, Estacionariedad, ACF, PACF, Modelo SARIMA, Modelo Logit, usando Python
